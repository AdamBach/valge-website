# Valge Website

Company website for Valge, a cross-border commerce operator connecting Korean tech and workspace products with global customers.

## Tech Stack

- Next.js
- React
- Tailwind CSS

## Local Development

```bash
npm install
npm run dev
```

Open http://localhost:3000.

## Production Build

```bash
npm run build
```

## Deployment

Deploy with Vercel, then connect the Cafe24 domain through DNS.
